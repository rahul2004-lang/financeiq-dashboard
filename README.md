# 💰 FinanceIQ Dashboard

A personal finance dashboard built with React, Zustand, and Recharts.

## 🚀 Features
- Dashboard (Income, Expense, Balance)
- Transactions (Add, Delete, Filter, Search)
- Insights (Pie + Bar Charts)
- Role-based access (Admin / Viewer)
- CSV Export
- Dark Mode
- Responsive Design

## 🛠 Tech Stack
- React
- TypeScript
- Zustand
- Recharts
- Tailwind CSS
